Download Star

A 2D Hypercasual Game Engine Written In JavaScript

Features

  1. 2D Collision Detection (Rect 2 Rect, Circle 2 Circle, Point To Circle, Point To Rectangle)
  2. A Base Game Class
  3. A Basic Event System
  4. Image/Sound/JSON Preloader
  5. Utility + Tiny Math Library (randomInt, random floats, shuffle array, etc.)
  6. Input Handling
  7. Device info
  8. DOM Manipulation
